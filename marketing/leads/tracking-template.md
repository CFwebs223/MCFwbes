# MCF.webs — Lead Tracking System

## Lead Status Pipeline
```
Cold → Contacted → Responded → Interested → Proposal → Negotiation → Won/Lost
```

## Lead Spreadsheet Structure

Create this in Google Sheets or Airtable.

| Column | Description |
|--------|-------------|
| Business Name | Name of the business |
| Industry | Restaurant, e-commerce, service, etc. |
| Website URL | Current website link |
| Website Quality | 1-5 rating (1 = outdated, 5 = premium) |
| Contact Name | Owner/decision maker |
| Phone | Direct line |
| Email | Business email |
| Instagram | @handle |
| LinkedIn | Profile URL |
| Source | Where you found them |
| First Contact Date | Date of first outreach |
| Outreach Method | DM / Email / Call |
| Status | Cold → Contacted → ... |
| Last Touchpoint | Date of last interaction |
| Notes | Pain points, budget signals, objections |
| Estimated Value | R2k / R5k / R8k |
| Follow-up Due | Next action date |
| Assigned To | Self |

## Lead Sources (Daily)

### Google Maps
Search: "restaurants cape town" "cafes near me" "gyms in johannesburg"
Look for: Sites with poor mobile, old designs, no booking system

### Instagram
Search hashtags: #capetownsmallbiz #saentrepreneur #localbusiness
Look for: Link in bio = Wix/Squarespace → DM opportunity

### LinkedIn
Search: "founder" + "restaurant" "owner" + "boutique" "CEO" + "wellness"
Connect + send personalized message

### Facebook Groups
Join: "South African Entrepreneurs" "Cape Town Business Owners"
Engage authentically, offer value, then DM

### Yellow Pages / SA Business Directories
Search for local businesses without websites or with poor ones
